# 🎓 University Club Information Website

This project is a collaborative web development assignment that demonstrates the use of Git and GitHub workflows in a team environment. The website provides information about university student clubs, events, and participation opportunities.

---

## 📁 Project Structure

student-portfolio/
├── .gitignore  
├── README.md  
├── src/  
│   ├── index.html  
│   ├── clubs.html  
│   ├── events.html  
│   ├── join.html  
│   └── highlights.html  
└── styles/  
    └── styles.css  

---

## 🌐 Features

- Home page introducing university clubs
- Club categories with descriptions
- Upcoming events section
- Guide on how to join clubs
- Highlights and achievements of clubs
- Responsive design using CSS media queries

---

## 🔀 Branching Strategy

- **production** → Stable, production-ready code  
- **develop** → Active development branch  
- **feature/** → Individual feature branches for each page  
- **release/v1.0** → Final testing before production  

---

## 👥 Team Roles

| Member        | Responsibility        |
|--------------|---------------------|
| Team Lead    | Home Page (index.html) |
| Member 1     | Clubs Page           |
| Member 2     | Events Page          |
| Member 3     | Join Page            |
| Member 4     | Highlights Page      |

---

## ⚙️ Git Workflow

1. Created feature branches from `develop`
2. Developed pages independently
3. Made multiple commits with proper messages
4. Created Pull Requests for each feature
5. Rebasing performed before merging
6. Merge conflicts resolved when required
7. Final release created and merged into `production`

---

## 🚀 How to Run

1. Clone the repository  
2. Open `src/index.html` in a browser  
3. Navigate through pages using links
